Команды и права	
Команды:
/abh reload - Перезагрузка конфига
/abh help - Посмотреть доступные команды
/abh enable - Включить систему анти-бхопа
/abh disable - Выключить систему анти-бхопп

Права:
abh.help
abh.enable
abh.disable
abh.reload
abh.access - даёт иммунитет к анти-бхопу
